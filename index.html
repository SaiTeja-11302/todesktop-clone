How it works

Step 1

Bootstrap straight from your web app

Copy and paste your web app url into ToDesktop. Customise your app design, app icon and window frame UI with no code.

  • Multiple windows
  • Customisable menus
  • Menubar/tray menus
  • Launch on startup
  • Offline support
  • Tabs (Mac only)
Step 2

Add desktop code to customise your web app

No need to create a new repository, just add code from our desktop APIs to your codebase and get access to system level functionality.

  • Multiple windows
  • Customisable menus
  • Menubar/tray menus
  • Launch on startup
  • Offline support
  • Tabs (Mac only)
Step 3

Publish

No need to create a new repository, just add code from our desktop APIs to your codebase and get access to system level functionality.

  • Multiple windows
  • Customisable menus
  • Menubar/tray menus
  • Launch on startup
  • Offline support
  • Tabs (Mac only)
